Overview: NHS 111 Domain Message Specification

The NHS 111 Domain Message Specification supports the following care communications:


  • NHS 111 CDA Document
  • NHS 111 Ambulance Request
  • NHS 111 Repeat Caller Request
  • NHS 111 Repeat Caller Response
  • Interaction to communicate Repeat Caller reports to a GP

The payloads contained in this specification are "universal". This means that the receiver does not need to have connectivity to the Spine Directory Services (SDS) to be able to fully process payloads. For further details on the use of universal messaging please refer to the Technical Guidance for Templated CDA Domains included in this specification (under the "Message Creation Guidance" tab). Also included in this Domain Message Specification are artefacts required to instantiate the message specifications.

These include:

  • Generic CDA Specification - for details click on the "Associated Specifications" tab
  • Nullification Implementation Manual - for details click on the "Associated Specifications" tab
  • Data Types Specification
  • Vocabulary Specification for Vocabularies used by this domain
