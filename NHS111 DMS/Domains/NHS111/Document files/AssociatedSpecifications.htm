Associated Specifications


Specification Description
Generic CDA The Generic CDA Domain Message Specification contains details on the "on-the-wire" format for CDA payloads. The "on-the-wire" format is the actual payload structure that is sent to a recipient. The Generic CDA Domain Message Specification documents two "on-the-wire" CDA formats; one for use over the Transaction Messaging System (TMS) when using TMS wrappers, and one for non-TMS usage. The NHS 111 Domain Message Specification uses the latter format, which is documented in the Generic CDA Domain Message Specification as "NHS CDA RMIM - POCD_MT000002UK01"
Nullification The Nullification Domain Message Specification contains details on how to nullify a CDA document that was, for example, sent in error or a document that is not accepted by a recipient. The domain specifies the message definition of the nullification document ("Nullification Document - POCD_MT130001UK05"), which is generically used to nullify any CDA document.
