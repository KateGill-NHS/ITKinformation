COCD_TP146245GB01
ObservationMedia

A template to allow attachments to be carried in a CDA document

This class is a derivative of the RIM Observation class that represents multimedia that is logically part of the current document. This class is only for multimedia that is logically part of the attested content of the document. Rendering ObservationMedia requires a software tool that recognizes the particular MIME media type.
