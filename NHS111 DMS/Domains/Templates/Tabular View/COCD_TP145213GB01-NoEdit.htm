RelatedSubject

A template to identify the subject of an attachment when it is not the patient

This class carries details of the subject of the attachment

Used by:
[1..1] [M] classCode (CS {CNE:x_DocumentSubject } )

indicates that the RelatedSubject role is the x_DocumentSubject.

Defaults to personal relationship (PRS).

[0..*] addr (SET<AD>)

A set of addresses associated with the role.

There are two optional additional components:

  • The use attribute should be used to identify the type of address
  • The useablePeriod child element may be used
[0..1] code (CE {CWE:PersonalRelationshipRoleType } )

The character of the relationship is defined by the PersonalRelationshipRoleType code

Any code system may be used for this attribute

The following attributes must be populated when this code is used
  • The code attribute shall contain a code from the relevant vocabulary
  • The codeSystem attribute shall contain the OID for the vocabulary
  • The displayName attribute shall carry a description associated with the code in the vocabulary
