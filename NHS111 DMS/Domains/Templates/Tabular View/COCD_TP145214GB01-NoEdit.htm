COCD_TP145214GB01
AssociatedEntity

A template to identify a document participant when no specific template exists.

this class identifies the role of the person associated with the document.

Used by:
[1..1] [M] classCode (CS {CNE:RoleClassAssociative } )
A code from the HL7 vocabulary domain to identify the type of associative role of the person.
[0..1] addr (AD)

One or more addresses associated with the role.

There are two optional additional components:

  • The use attribute should be used to identify the type of address
  • The useablePeriod child element may be used

[0..1] code (CV {CWE:RoleCode } )

A code the further defines the code carried in the classCode attribute to allow a more detailed identification of the type of role the person plays as a document participant.

The Coded Plain flavour of Coded Value (CV) datatype shall be used to carry this information as follows:

  • The code attribute shall contain the value
  • The codeSystem attribute shall contain the OID for the vocabulary concerned
  • The displayName attribute shall contain the text associated with the code

[0..*] id (SET<II>)

A valid identifier that uniquely identifies the associative role.

  • The root attribute shall contain an OID
  • The extension attribute shall contain the identifier.
[0..*] telecom (SET<TEL>)

One or more telecoms associated with the role.

There are two optional additional components:

  • The use attribute may be used to identify the type of telecom
  • The useablePeriod child element may be used

[1..1] [M] templateId (II)
{ Fixed="COCD_TP145214GB01#AssociatedEntity" }

The fixed value of this attribute provides a unique identifier for the template and the classname within that template.

The IdentifierExternal datatype flavour is used to carry this information as follows:

  • The root attribute shall contain the OID "2.16.840.1.113883.2.1.3.2.4.18.2"
  • The extension attribute shall contain the fixed value.
[0..1] associatedPerson (Person)

A link to the person entity which is playing this role.
[0..1] scopingOrganization (Organization)

A link to the scoping organisation.

Organization

Details to allow identification of the organisation.

Used by: AssociatedEntity
[1..1] [M] classCode (CS {CNE:EntityClass } )
{ Fixed="ORG" }

Indicates this is an organisation.
[1..1] [M] determinerCode (CS {CNE:EntityDeterminer } )
{ Fixed="INSTANCE" }

Indicates this is an instance of an organisation.
[1..1] [M] id (II)

A valid identifier that uniquely identifies the organisation.

The organisation id attribute can be populated using the following ODS codes (formerly known as NACS code) :

  • The root attribute shall contain an OID with the value "2.16.840.1.113883.2.1.3.2.4.19.2"
  • The extension attribute shall contain the ODS site code.
  • The root attribute shall contain an OID with the value "2.16.840.1.113883.2.1.3.2.4.19.1"
  • The extension attribute shall contain the ODS organisation code.

Note these are listed in order preference for populating this attribute.
[1..1] [M] name (ON)
