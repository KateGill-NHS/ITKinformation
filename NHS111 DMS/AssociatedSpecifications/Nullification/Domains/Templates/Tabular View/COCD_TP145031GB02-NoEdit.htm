COCD_TP145031GB02
WorkgroupSDS

A template to identify a workgroup or team and a person within the workgroup or team.

This class carries details of the workgroup or team

Used by:
[1..1] [M] classCode (CS {CNE:RoleClass } )
{ Fixed="ASSIGNED" }
Indicates that this role is assigned.
[1..1] [M] code (CV {CNE:WorkgroupRoleName } )

The purpose of this attribute is to hold the workgroup name in the displayName attribute.

The Coded Plain flavour of the Coded Value (CV) data type shall be used to carry this information as follows:

  • The code attribute shall contain the fixed value "01"
  • The codeSystem attribute shall contain the OID "2.16.840.1.113883.2.1.3.2.4.17.266"
  • The displayName attribute shall carry the name as defined in SDS

Where no workgroup name is available a null flavour of "NI" (no information) shall be used as show

<nullFlavor="NI"/>

[1..1] [M] id (II)

An SDS identifier that identifies a workgroup or team.

  • The root attribute shall be an OID with the value 1.2.826.0.1285.0.2.0.109
  • The extension attribute shall be the SDS workgroup code.
