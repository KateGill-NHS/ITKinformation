Vocabulary for WorkgroupRoleName

2.16.840.1.113883.2.1.3.2.4.17.266

Version: 1.0
Status: active
Date: 2009-03-09

Description

This vocabulary is the default one which contains no values and its only purpose is to allow the XML attribute displayName to carry the name of the workgroup or team.

Please refer to the tabular view for guidance on how to code workgroup role names.
