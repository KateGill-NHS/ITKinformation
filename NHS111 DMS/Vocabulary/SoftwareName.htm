Vocabulary for SoftwareName

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.333

Version: 1.0
Status: active
Date: 2012-01-30

Description

  • This vocabulary is used to describe the software installed / running on the device or system.
  • This is a HL7 vocabulary and there are currently no codes defined by HL7 to describe software.
  • Please refer to the tabular view for guidance on how to code software.
