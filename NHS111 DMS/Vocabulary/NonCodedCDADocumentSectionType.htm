Vocabulary for NonCodedCDADocumentSectionType

Code System Id - An appropriate Id that identifies the coding system being used

Version: 1.0
Status: created
Date: 2011-05-17

Description

The coded attribute that uses this vocabulary is defined as CWE and therefore any allowed coding system may be used.

The coding system used must be the coding system used for the initial data capture. 

The use of the SNOMED CT coding system is NOT allowed when using templates that use this vocabulary, however any allowed coding system being used must use translation to show the SNOMED CT concept where an approved mapping exists.  

 The allowed coding systems, use of translation and mapping of these allowed coding systems will be defined  further in guidance to be issued in due course.
