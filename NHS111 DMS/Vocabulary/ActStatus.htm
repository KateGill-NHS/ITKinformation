Vocabulary for ActStatus

Code System Id

2.16.840.1.113883.5.14

Version: 1.0
Status: active
Date: 2012-02-24

Description

Codes representing the defined possible states of an Act.

Values

Code Description Note
aborted aborted

The Act has been terminated prior to the originally intended completion.
active active

The Act can be performed or is being performed.
cancelled cancelled

The Act has been abandoned before activation.
completed completed

The Act has been terminated normally after all of its constituents have been performed.
held held

The Act that is still in the preparatory stages has been put aside. No action can occur until the Act is released.
new new

The Act that is in the preparatory stages and may not yet be acted upon.
normal normal

This encompasses the expected states of an Act, but excludes "nullified" and "obsolete" which represent unusual terminal states for the life-cycle.
nullified nullified

This Act instance was created in error and has been 'removed' and is treated as though it never existed. A record is retained for audit purposes only.
obsolete obsolete

This Act instance has been replaced by a new instance.
suspended suspended

The Act that has been activated (actions could or have been performed against it), but has been temporarily disabled. No further action should be taken against it until it is released.
