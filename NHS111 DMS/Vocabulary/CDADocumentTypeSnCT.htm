Vocabulary for CDADocumentTypeSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 4.0
Status: active
Date: 2012-01-31

Description

When using a single SNOMED CT code that describes a type of CDA document then:

  • Any code from the SNOMED CT UK Document type subset with subset original id 1061000000133
  • The OID to use is 2.16.840.1.113883.2.1.3.2.4.15


When using a compositional expression consisting of multiple codes from SNOMED CT that describes a type of CDA document then:

  • A code from the SNOMED CT UK Correspondence Document type subset with subset original id 44041000000135
  • A code from the SNOMED CT UK Correspondence CareSetting type subset with subset original id 43971000000130
  • The OID to use is 2.16.840.1.113883.2.1.3.2.4.15.1
  • Further guidance on the format of the compositional expression is here

A single code from a suitable alternative vocabulary:

  • When using an alternative vocabulary code, then the OID shall contain the OID of the alternative vocabulary
  • When no ID is available for the alternative vocabulary, then the OID "2.16.840.1.113883.2.1.3.2.4.17.337" shall be used
