Vocabulary for NHS111CDADocumentSectionType

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.323

Version: 1.0
Status: active
Date: 2011-11-17

Description

This vocabulary contains a set of codes to describe each text section in the NHS 111 document to provide a mechanism to allow receivers to process and reuse text sections.


Individual domains may specify the codes to use as follows:
  • For NHS 111 section documents there are currently no national codes defined yet
  • For local document section codes use the OID "2.16.840.1.113883.2.1.3.2.4.17.323"
